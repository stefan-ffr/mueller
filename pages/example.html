Beispielseite

Dies ist eine Beispielseite für interne Kacheln. Du kannst hier beliebigen Content hinzufügen.

So funktioniert es:

  1. Erstelle eine neue JSON-Datei in data/tiles/
  2. Füge die Tile-ID zum tiles-Array in data/shared.json hinzu
  3. Für interne Seiten: Erstelle eine HTML-Datei in pages/
  4. Für externe Links: Gib einfach die URL an

Beispiel-Kachel JSON:

{
  "id": "meine-kachel",
  "title": {
    "de": "Meine Kachel",
    "en": "My Tile",
    "th": "แผงของฉัน"
  },
  "description": {
    "de": "Beschreibung",
    "en": "Description",
    "th": "คำอธิบาย"
  },
  "icon": "🎨",
  "type": "internal",
  "url": "pages/meine-seite.html",
  "displayOrder": 102,
  "theme": {
    "gradientFrom": "blue-400",
    "gradientTo": "purple-600"
  }
}
